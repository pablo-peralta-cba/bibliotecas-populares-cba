# Popular Libraries of Córdoba, Argentina

This project provides a web interface to explore and discover popular libraries located in the province of Córdoba, Argentina. It aims to connect users with these valuable community resources.

## Features

- Browse Libraries: Easily view a list of popular libraries in Córdoba.
- View Details: Access key information about each library, such as address, contact details and book catalogs.
- User authentication via Passport and Nodemailer.
- Provide comments and reviews on libraries profiles.
- Create and own new libraries profiles and update its data.
- Search Functionality: Quickly find libraries based on name or keywords.
- Interactive Map: Visualize library locations on a map.

## Installation and Running Locally

To get a local copy of the project up and running, follow these simple steps:

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/pablo-peralta-cba/bibliotecas-populares-cba.git](https://github.com/pablo-peralta-cba/bibliotecas-populares-cba.git)
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd bibliotecas-populares-cba
    ```
3.  **Install the dependencies:**
    ```bash
    npm install
    ```
4.  **Set up environment variables:**
    Create a `.env` file in the root directory and configure your environment variables (e.g., database connection string, API keys, etc.). Refer to the project's configuration for the required variables.
5.  **Start the server:**
    ```bash
    npm start
    ```

This command will start the Node.js server. You should then be able to access the application in your browser, likely at `http://localhost:3000` or a similar address depending on your configuration.
